# vtu-design-patterns

# About Application
The application register celebrities into chat room and then they can chat between each other.

## Implemented design patterns
* Creational patterns
	* Factory Method
	* Abstract Factory
* Structural patterns
	* Facade
* Behavioral patterns
	* Command
	* Mediator
